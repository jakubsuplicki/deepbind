# DeepFilesAI Sample Business Folder

This folder is fictional demo data for the source import flow. It is safe to
ship with the desktop app and contains no customer or third-party confidential
material.

Use it to demonstrate the same consent flow as a real folder:

1. Select the sample folder.
2. Scan file names, types, sizes, and folders.
3. Review what will be imported.
4. Approve the content import.
5. Ask questions about the created local memory.
