# OWASP Top 10 for Large Language Model Applications

The OWASP LLM Top 10 (v1.1) documents the most critical security risks associated
with deploying and using Large Language Model applications. It is the primary
reference for LLM application security practitioners.

poisoning, sensitive information disclosure, supply chain, excessive agency, overreliance.
