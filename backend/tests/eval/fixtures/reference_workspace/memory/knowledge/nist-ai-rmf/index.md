# NIST AI Risk Management Framework (AI RMF 1.0)

The NIST AI RMF provides guidance for organizations designing, developing, deploying,
evaluating, and acquiring AI systems to manage risk effectively. It is organized
around four core functions: **Govern, Map, Measure, Manage**.
