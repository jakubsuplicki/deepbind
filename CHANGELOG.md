# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-14

Initial public release.

DeepFilesAI is a pure-local AI workspace that remembers what matters: it
imports notes, files, PDFs, and URLs into a Markdown vault, retrieves with
hybrid keyword + semantic + graph search, reasons with a local model, and
writes useful outputs back into durable Markdown memory.
